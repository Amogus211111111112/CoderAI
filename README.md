CoderAI wurde auf Python programmiert

CoderAI verwendet Python Bibliotheken: 
                                      customtkinter  für fenster und GUI
                                      g4f   Ermöglicht die kostenlose Nutzung verschiedener KI-Modelle (hier GPT-4) durch Zugriff auf Drittanbieter-APIs
                                      json   für speichern des Chatverlaufs in  json Datei Format
                                      os   Wird für Operationen mit dem Dateisystem verwendet (Ordner erstellen, Dateien prüfen)
                                      uuid   Generiert eine eindeutige ID (user_code) für Benutzer
                                      datetimt  Wird benötigt, um den Zeitstempel beim Speichern von Chat-Nachrichten festzuhalten

Das schwirigste war es das verhalten des KI Agenten einzustellen

Es wird empfohlen denn KI Agenten nicht zu sehr viel zu schreiben, weil OpenAi kann denn Zugriff drauf verweigern

um denn KI Agenten zu starten starte die Datei app.py
